# 🚀 Portfolio 2026

A modern and clean single-page portfolio built with React + Vite.

---

## 📦 Tech Stack

* React (Vite)
* Tailwind CSS
* Framer Motion
* React Icons
* Vercel Analytics

---

## ⚙️ Getting Started

### Lighthouse

```bash
lighthouse http://localhost:5173/
```

---


### 1. Clone the repository

```bash
git clone https://github.com/your-username/portfolio_2026.git
cd portfolio_2026
```

---

### 2. Install dependencies

```bash
npm install
```

---

### 3. Run the project

```bash
npm run dev
```

---

## Build for production

```bash
npm run build
```
